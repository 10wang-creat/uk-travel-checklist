🇬🇧 英國16天旅遊打包清單

⚠️ 最重要：防水外套、雨傘、保暖衣物必備！
📅 日期：10/10-10/25（16天）
🌡️ 氣溫：9-18度C

📦 行李箱配置

✅ 可以帶7個28吋行李箱

1 每人一個28吋行李箱
2 只裝70-80%滿
3 預留空間給購物

👕 每人必帶衣物（4-5天份）

  • 發熱衣 x 2件
  • 長袖上衣 x 2件
  • 薄毛衣 x 1件
  • 厚外套 x 1件
  • 防水外套 x 1件（必備！）
  • 內衣褲 x 5套
  • 襪子 x 5雙

👗 女生特別準備

  • 長裙 x 2-3條
  • 厚褲襪 x 3-4雙（120丹以上）
  • 備用長褲 x 1條
💡 長裙+厚褲襪好處：
• 保暖度跟長褲一樣
• 上廁所更方便
• 更省行李空間

👞 鞋子（每人2雙）

  • 防水健走鞋 x 1雙（主要穿）
  • 休閒鞋/運動鞋 x 1雙（輪替）
  • 室內拖鞋 x 1雙

🧣 保暖配件（每人）

  • 毛帽 x 1頂
  • 手套 x 1雙
  • 圍巾 x 1條
  • 雨傘 x 1把

💊 醫藥包

  • 感冒藥、退燒藥

  • 暈車藥（山路多）
  • OK繃、優碘

  • 車充 x 3個
